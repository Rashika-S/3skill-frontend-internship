# 🏹 Baahubali Quiz Arena

> **"How well do you know the kingdom of Mahishmati?"**

An interactive fan-made quiz application inspired by the epic **Baahubali** universe. Challenge yourself with multiple-choice questions about legendary characters, unforgettable battles, iconic dialogues, and the rich history of Mahishmati.

This project was built using **Lovable AI** as part of my **Frontend Development Internship**, showcasing how AI-assisted development can rapidly transform an idea into a fully functional web application.

## 🌐 Live Demo

🔗 **https://baahubali-quiz.lovable.app**

---

## 🎬 Inspiration

The idea for this project came after watching the **Baahubali documentary/movie**, which inspired me to create an engaging fan experience that lets users relive the world of Mahishmati through an interactive quiz.

---

## 🎯 Project Goal

To create an immersive and enjoyable experience for Baahubali fans by combining storytelling with an interactive quiz interface while exploring AI-assisted web development using **Lovable AI**.

---

## ✨ Features

* 📝 Multiple-choice quiz questions
* ⏱️ Countdown timer
* 📊 Real-time score tracking
* 📱 Fully responsive interface
* ⚡ Fast and interactive user experience
* 🎨 Clean, modern UI
* 🔄 Replay the quiz anytime

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/Rashika-S/3skill-frontend-internship.git
```

### Navigate to the project folder

```bash
cd baahubali-quiz
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```
