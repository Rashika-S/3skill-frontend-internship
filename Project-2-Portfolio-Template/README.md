# 🦋 Elegant Portfolio Template

## Live Demo

🔗 https://project-2-ten-wheat.vercel.app/

## Overview

A clean, feminine, modern portfolio website template built with HTML, CSS and Tailwind CSS (CDN only). Designed to be reusable by students, designers, developers and professionals by simply replacing content, images and links.

## Technologies Used

* HTML5
* CSS3
* Tailwind CSS
* JavaScript

## Features

* Multi-page Portfolio
* Responsive Design
* Animated Butterfly Elements
* Glassmorphism Cards
* Certificates Showcase
* Skills Categorization
* Project Showcase
* Contact Page
* Resume Integration
