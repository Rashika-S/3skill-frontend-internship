# 🎉 EventNest
*"Discover. Plan. Celebrate."*

A modern event management web application designed to help users explore, discover, and organize events with ease. EventNest provides an intuitive interface for browsing upcoming events, viewing event details, and creating a seamless event discovery experience.

This project was built using **Lovable AI** as part of my **Frontend Development Internship**, showcasing how AI-assisted development can transform an idea into a responsive and feature-rich web application.

## 🌐 Live Demo
🔗 https://project-5-eventnest.lovable.app

## 🎯 Inspiration
The inspiration behind EventNest was to create a platform that simplifies event discovery while providing users with an engaging and modern browsing experience. The goal was to design a clean interface that makes finding and exploring events effortless.

## 🎯 Project Goal
To build a responsive event management platform that demonstrates modern frontend development practices while exploring AI-assisted development with Lovable AI.

## ✨ Features
📅 Browse upcoming events

🔍 Search and discover events

🗂️ Category-based event organization
